📊 Financial Modeling of Infosys Ltd.

🏢 Overview

This repository contains an Excel-based financial model of Infosys Ltd., developed as part of my MBA (Finance) curriculum. The project focuses on analyzing historical financial performance, building key assumptions, forecasting future financials, and performing basic valuation.

🎯 Objectives

To develop practical skills in financial modeling
To analyze company financial statements
To perform forecasting using Excel
To understand basic valuation techniques

📈 Key Features

Analysis of Income Statement, Balance Sheet, and Cash Flow
Revenue and expense forecasting
Assumptions-based financial projections
Basic Discounted Cash Flow (DCF) valuation
Structured and linked Excel model

🛠 Tools Used

Microsoft Excel

💡 Key Learnings

Gained hands-on experience in financial modeling
Improved understanding of FP&A concepts (forecasting & analysis)
Strengthened Excel and financial analysis skills

⚠️ Disclaimer

This project is created for academic purposes only and is not affiliated with or endorsed by Infosys Ltd. All data used is sourced from publicly available information.
